This will install Elite Bulletin Board version 3.0.0 Release Candidate 1. Once this is complete, you will be able to enjoy this application on your website.

If you already installed a previous version of version 3, click on upgrade from v3. If you're new to EBB, click on Fresh Install, both links found above.

Upgrading means you will lose some pre-defined values, you will be able to re-configure it later after the upgrade.

If you are unsure about using this version, please download the stable release.
